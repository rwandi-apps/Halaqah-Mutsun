
# SDQ Management System

Portal Management untuk SDQ yang terintegrasi dengan AI untuk evaluasi perkembangan siswa.

## Features
- Dashboard Koordinator & Guru
- Manajemen Data Guru, Siswa, dan Kelas
- Laporan Halaqah Real-time
- AI Feedback menggunakan Gemini API
- PWA Support

## Tech Stack
- React 18
- Tailwind CSS
- Firebase (Auth & Firestore)
- Gemini SDK (@google/genai)
